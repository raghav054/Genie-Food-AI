# Genie Food

Genie Food is a full-stack food ordering web application developed using the MERN stack. The project allows users to browse restaurants, explore menus, place online orders, and make secure payments. It also integrates AI to generate food descriptions and summarize restaurant reviews, helping users make better ordering decisions.

## Project Overview

This project was built to provide a complete online food ordering experience while exploring modern web development technologies and AI integration. It focuses on clean UI, secure authentication, payment processing, and RESTful API development.

## Features

- User registration and login using JWT authentication
- Browse restaurants and food menus
- Search restaurants
- Add and manage items in cart
- Secure online payments using Stripe
- Order history and order management
- Restaurant ratings and reviews
- AI-generated food descriptions
- AI-generated restaurant review summaries
- Responsive user interface
- Cloudinary image upload support
